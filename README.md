# 📖 **Project Overview**

This project leverages **Artificial Intelligence** to analyze, review, and suggest improvements for code, enhancing development efficiency.  
It’s a great project to learn **Full-Stack Development with AI integration**.

---

# 🚀 **Features & Implementation**

- **Intro & User Flow** – A simple and intuitive flow with live preview.  
- **Backend Setup** – Configured a Node.js server with Express.  
- **AI Integration** – Added AI response feature using Google Generative AI.  
- **Refactored API Handling** – Improved request structure and switched to POST method for better reliability.  
- **Frontend with React + Vite** – Initialized a React project using Vite with clean styling.  
- **Syntax Highlighting** – Integrated PrismJS for beautiful code highlighting.  
- **CORS & Axios** – Enabled CORS support and used Axios to connect frontend and backend seamlessly.  
- **Code Review Rendering** – Enhanced the review feature with Markdown rendering and styling improvements.
